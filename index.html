원칙 · Rules
  • 굵기는 400 / 500 / 600 / 700 4단계만 사용. 그 외 weight 금지.
  • 제목(Heading)은 600~700, 본문(Body)은 400으로 위계를 만든다.
  • line-height는 제목 130%, 본문 170%, 캡션 150%로 고정.
  • 한글 자간(letter-spacing)은 -1% 기본, 20px 이상 큰 제목은 -2%.
  • 소수점 폰트 크기(13.97px, 15.96px 등) 사용 금지 — 위 정수 토큰만 사용.
  • 반응형은 요소별 임의 배율 대신 base + ≥1600px(2xl) 토큰 한 쌍으로 통일.

적용 · Tailwind
CSS에 한 번 정의하면 text-body처럼 base·2xl이 같이 적용됩니다.
@layer components {
  .text-display    { @apply text-[24px] 2xl:text-[30px]; }
  .text-heading1   { @apply text-[18px] 2xl:text-[24px]; }
  .text-heading2   { @apply text-[15px] 2xl:text-[20px]; }
  .text-heading3   { @apply text-[14px] 2xl:text-[18px]; }
  .text-heading4   { @apply text-[13px] 2xl:text-[17px]; }
